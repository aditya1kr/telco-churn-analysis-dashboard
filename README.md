Telco Customer Churn Analysis Dashboard

Project Overview
This project analyzes customer churn behavior in a telecom company using Power BI and SQL. The goal is to identify key factors driving churn and provide actionable business insights.

Objectives
- Analyze customer churn rate
- Identify high-risk customer segments
- Understand revenue impact of churn
- Provide business recommendations

Tools & Technologies
- Power BI
- SQL
- DAX
- Power Query

Dataset Features
- Customer ID
- Gender
- Tenure
- Contract Type
- Internet Service
- Payment Method
- Monthly Charges
- Total Charges
- Churn

Dashboard Pages
 Executive Overview
- KPIs: Total Customers, Revenue, Churn Rate
- Customer distribution
- Gauge chart for churn vs target

 Churn Analysis
- Churn Rate by Contract
- Churn by Tenure & Services
- Decomposition Tree for root cause

 Business Impact
- Revenue analysis
- Revenue loss due to churn
- Waterfall chart
- Top customers
- Customer segmentation (Scatter plot)

Key Insights
- Churn rate is **26.5%**, significantly higher than benchmark
- Month-to-month customers have highest churn
- New customers are more likely to churn
- Revenue loss is directly impacted by churn

Business Recommendations
- Promote long-term contracts
- Improve onboarding for new customers
- Target high-risk segments
- Enhance service quality

Future Scope
- Predictive churn modeling
- Real-time dashboard integration
